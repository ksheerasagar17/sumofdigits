# sumofdigits

## Purpose
This program automatically generates sum of all the digits in between two numbers (including) that are equal to a number.

### For example, the below inputs will generate
```
var from = 1200;
var to = 1259;
var increment = 1;
var sumEquals = 3;
```

### Output
***1200***<br />
***1209***<br />
***1218***<br />
***1227***<br />
***1236***<br />
***1245***<br />
***1254***<br />

## Build
```
cd src; 
dotnet build;
```
## Run
```
cd src; 
dotnet run;
```
